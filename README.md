Instructions to run & execute C programs in FEDORA 18:

1. Open TERMINAL

    Applications -> SystemTools -> Terminal

2. To create a C file

    $ gedit filename.c
    
3. To compile & check the errors
    
    $ cc filename.c

4. To Check results/output

    $ ./a.out
